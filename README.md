# Portfolio

A personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**, showcasing my projects, skills, and tech stack.

## Features

- Fully responsive design  
- Animated sections with `motion/react`  
- Tech Stack showcase (Frontend skills)  
- Projects with images and descriptions  
- Smooth scrolling to sections  

## Tech Stack

- React (Vite)  
- Tailwind CSS  
- TypeScript  
- Motion for animations  

## Getting Started

```bash
npm install
npm run dev       # run locally
npm run build     # build for production
npm run preview   # preview production build locally
